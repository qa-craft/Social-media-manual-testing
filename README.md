# Social Media Manual Testing Project

This repository demonstrates manual testing of a social media web application.

The objective of this project is to simulate real-world QA documentation and testing activities including requirement analysis, test design, execution, and bug reporting.

---

# Application Tested

Social Media Web Application

---

# Modules Covered

Login Module
Profile Module
Post Creation Module
News Feed Module
Messaging Module

---

# Testing Types Performed

Functional Testing
UI Testing
Negative Testing
Usability Testing

---

# Testing Artifacts Included

The project includes the following QA documentation:

• Requirements
• Test Plan
• Test Scenarios
• Test Cases
• Test Execution Report
• Bug Report Log
• Requirement Traceability Matrix (RTM)

All detailed testing documents are available in the **Test_Documents** folder.
